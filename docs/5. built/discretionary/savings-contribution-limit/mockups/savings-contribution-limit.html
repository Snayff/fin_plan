Pattern compliance notes
· Nudge appears in the right panel only as a NudgeCard — never as an inline tag on rows (anchor #13).
· Row dot reuses the existing attention amber pattern — single colour, single meaning ("noteworthy"). Disambiguated by the right panel.
· Language is arithmetic-only: spare amount, two rates, annual £ uplift. No "should", no recommendations (anchor #9).
· Over-cap is informational, never blocks save (anchor #12).
· The capacity meter uses tier-discretionary purple normally (Savings is a discretionary destination) and switches to attention amber only when over-cap.
· Form field hidden unless type === "Savings".
