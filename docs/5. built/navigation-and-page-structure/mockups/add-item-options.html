My recommendation: C (Header-aligned button)
It's discoverable without dominating. The header is already a "chrome" zone — the button lives with the title, not with the data. Ghost border at rest means it reads as part of the heading until you need it. It also scales well: if we later add other header actions (sort, filter), they sit naturally alongside.
