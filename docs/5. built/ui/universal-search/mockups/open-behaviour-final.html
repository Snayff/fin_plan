Open behaviour — anchored drop vs expand-over-nav

Position locked at R. Two visibly-different open behaviours compared side-by-side.

E

Anchored drop

The nav row stays put. A full card (input + results) drops from under it. Dimming happens only in the body area. The chrome is untouched.

F

Expand over nav

The icon morphs into a full-width search input that covers the nav row. Brand stays on left, household switcher and avatar stay on right. Results drop below the expanded input. Closing the search restores the nav.
