# Mobile Experience

## Intention
FinPlan is desktop-first, but users occasionally need to check their financial plan on a phone. A minimal mobile view for the Overview allows quick reference without requiring a full responsive redesign.

## Description
The Overview page collapses the two-panel layout to a single column of cards on small screens. Full editing and navigation remain desktop-only. Scope and design not yet defined.

## User Stories
- As a mobile user, I want to view my waterfall on my phone so that I can check my finances quickly when away from a desktop.

## Acceptance Criteria
- [ ] Not yet designed — full scope TBD
- [ ] Overview page is usable on a small screen as a minimum
- [ ] Two-panel layout collapses to single column on mobile breakpoint
