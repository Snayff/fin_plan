Doughnut Segment Colour Palettes

Three approaches for generating up to 7 visually distinct segments per tier. Shown for both Committed (indigo) and Discretionary (purple).

COMMITTED
DISCRETIONARY

A — Opacity Variations

Same hex colour at decreasing opacities (100% → 14%). Strictly within the tier identity. Smaller segments fade into the dark background — potentially hard to read.

COMMITTED
DISCRETIONARY

B — Lightness / Saturation Variations

Steps through the Tailwind indigo and purple scales (light → dark). Every segment is a distinct solid colour, all within the same hue family. More visually distinct than opacity, stays on-brand.

COMMITTED
DISCRETIONARY

C — Neutral Slate Palette

Slate greys for both tiers. No tier colour bleed risk, but both doughnuts look identical — the tier identity comes only from the heading, not the chart itself.
