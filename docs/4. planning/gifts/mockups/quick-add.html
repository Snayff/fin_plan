How it works

  • Matrix cells are editable inputs — tab across to fill, empty = no gift planned
  • Row totals on the right, column totals at the bottom
  • "+ add person" row at the bottom of the table inserts a new person row inline
  • "+ event" column header inserts a new event column inline
  • Live summary strip below the matrix shows Budget / Currently allocated / Remaining as cells are edited
  • Save commits all changes (creates new people/events, updates allocations). Cancel discards, returning to Config State 2.
  • Pre-populated on open — existing people and current-year allocations are loaded, so Quick Add is also a bulk-edit tool, not just bulk-create
  • Dates, notes, status are not editable here — only in the per-person detail view (Gifts mode). Items created here are flagged "needs date" on dateless events.
